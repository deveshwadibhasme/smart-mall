## Application Flow

1. User registers or logs in
2. User browses products
3. User places an order
4. System generates secure pickup code
5. Inventory updates automatically
6. User visits store
7. Staff verifies pickup code
8. Order marked as completed

## Problem Solved

SmartMall addresses key challenges in physical retail operations:

- Reduces long billing queues through online ordering
- Minimizes counter congestion with secure pickup system
- Improves inventory accuracy with real-time stock updates
- Ensures secure order collection via unique pickup codes
- Enhances store management through centralized admin control
- Supports digital transformation of traditional retail stores
